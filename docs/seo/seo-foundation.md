# SEO Foundation

## Indexable page classes

- Homepage
- Section landing pages
- category/location pages
- listing detail pages
- business profile pages
- company profile pages
- project/building/model pages
- help/blog/editorial support pages

## Noindex candidates

- low-value filtered duplicates
- empty results pages
- internal workspaces/admin pages
- staging domain

- sitemap index
- canonical generation
- breadcrumb schema
- organization schema
- job posting schema
- hreflang-ready routing
